## Q & A
Q: 为什么存在两种图片格式？

A: 因为将脑图绘制工具从 MindManager 换为了 MindNode

Q: 脑图包含了对应书籍的完整内容吗？

A: 大部分是，但少部分因为兴趣和理解的原因有所取舍
